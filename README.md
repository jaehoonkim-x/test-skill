# 쿠팡 리뷰 스크래퍼 플러그인

쿠팡에서 키워드로 상품을 검색하고, 검색 결과의 각 상품 페이지를 방문하여 리뷰 수를 수집하는 자동화 플러그인입니다.

## 구성 요소

| 구성 | 설명 |
|------|------|
| 스킬: coupang-review-scraper | 쿠팡 스크래핑 워크플로우 가이드 (자동 트리거) |
| 커맨드: /coupang-review | 키워드를 지정하여 바로 리뷰 수집 실행 |

## 사용법

### 커맨드로 실행

```
/coupang-review 장갑
```

키워드를 인자로 전달하면 즉시 해당 키워드로 쿠팡 검색 및 리뷰 수집을 시작합니다.

### 자연어로 실행

"쿠팡에서 텀블러 리뷰 수 수집해줘" 같은 자연어 요청으로도 스킬이 자동 트리거됩니다.

## 요구사항

- Playwright MCP가 연결되어 있어야 합니다
- 브라우저 자동화 도구 (browser_navigate, browser_evaluate, browser_run_code 등) 사용 가능해야 합니다

## 출력 예시

| 순위 | 상품명 | 리뷰 수 |
|------|--------|---------|
| 1 | 코멧 니트릴장갑 블랙 | 50,951 |
| 2 | 탑클린 니트릴 장갑 | 32,100 |
| 3 | 크린가드 니트릴 장갑 | 28,450 |

## 참고

- 쿠팡은 HTTP 직접 요청을 차단하므로 반드시 Playwright 브라우저를 통해 접근합니다
- 타임아웃 방지를 위해 상품을 9개씩 배치로 처리합니다
- 쿠팡 DOM 구조 변경 시 셀렉터 업데이트가 필요할 수 있습니다
